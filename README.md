# ♟️ Chess 
A browser-based chess application built with **TypeScript** and **HTML5 Canvas**, designed with a strong emphasis on clean, class-based architecture and separation of concerns. 
The project is structured around clearly defined modules for game state, rendering, and input handling. 
Core systems are implemented using object-oriented principles to ensure scalability, maintainability, and extensibility as features evolve. 
This is an ongoing project aimed at demonstrating thoughtful software design, modular architecture, and real-time rendering patterns in a front-end environment.
